# OpenSpec 指令

这些指令用于在此项目中工作的 AI 助手。

当请求符合以下情况时,请始终打开 `@/openspec/AGENTS.md`:
- 提到规划或提案(如 proposal、spec、change、plan 等词)
- 引入新功能、破坏性变更、架构调整或重大性能/安全工作
- 听起来模棱两可,你需要在编码前查看权威规范

使用 `@/openspec/AGENTS.md` 来学习:
- 如何创建和应用变更提案
- 规范格式和约定
- 项目结构和指南

保留此托管块,以便 'openspec update' 可以刷新指令。
